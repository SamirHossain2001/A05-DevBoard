Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select an element in the DOM. The most common methods are:
1. Select an element by id using getElementById() function.
2. Select multiple element having a common class using getElementsByClassName() function.
3. Select multiple elements by their tag name using getElementsByTagName() function. 4. Select multiple elements using CSS selector using querySelectorAll() function.
5. Select the first element that matches the CSS selector using querySelector() function.

Question-2: What is the difference between innerHTML, innerText, and textContent?

In HTML, innerHTML, innerText, and textContent are used to read and update the content of HTML elements.
innerHTML: it can read and update both HTML tag and text content.
innerText: it can only read and update the text inside an element. It also ignore hidden content.
textContent: it is same as innerText but the main difference is it can read and update the hidden contents.

Question-3: What is event delegation in the DOM?

When we have multiple child inside a single container, adding event listener to all of them individually makes it slower to process. Also if we want to add a new child, we have to add an event listener every time. Event delegation is a tecnique where single parent handle event of multiple child using bubbling tecnique. By using event.target we can capture any click events.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in which an event begins at the target element and travels upward through its parent elements in the DOM hierarchy.
Like: body -> div -> button, if the button is being clicked, it will capture button first using capture phase. Then it will bubble upward div -> section -> body -> HTML -> document.

Question-5: How do you create, add, and remove elements using JavaScript?

create: we can create an element using document.createElement(tag) function.
add: we can append a new element using parentContainer.appendChild(element) function.
remove: we can remove an element using parentContainer.removeChild(element) or child.remove() function.
